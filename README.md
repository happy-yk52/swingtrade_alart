# ◈ SWING-AI

日本株スイングトレード（順張りトレンドフォロー）向けテクニカル分析ダッシュボード。

監視銘柄のテクニカルシグナルをスマホで毎日チェック。気になる銘柄はワンタップで Claude に分析を依頼できます。

## 特徴

- **20銘柄の一括監視** — MA / RSI / MACD / BB / 出来高の5指標でスコアリング
- **買い・売りシグナルを自動判定** — スコア -10〜+10 の7段階評価
- **エントリー4条件チェック** — 順張りスイングの基本条件を自動判定
- **Claude 連携** — 📋ボタンでプロンプトをコピー → Claude.ai に貼るだけ
- **完全無料運用** — yfinance + GitHub Actions + GitHub Pages（追加コストは Claude Pro のみ）

## セットアップ手順

### Step 1: GitHub にリポジトリ作成

GitHub で **Public** リポジトリを新規作成します。

### Step 2: ファイルをアップロード

以下のファイル構成でアップロードしてください：

```
リポジトリ/
├── watchlist.py
├── generate_data.py
├── README.md
├── .github/
│   └── workflows/
│       └── update.yml
└── docs/
    ├── index.html
    └── manifest.json
```

### Step 3: GitHub Pages を有効化

1. リポジトリの **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / フォルダ: **/docs**
4. **Save** をクリック

### Step 4: 初回データ生成

1. リポジトリの **Actions** タブを開く
2. 左メニューから **Update Stock Data** を選択
3. **Run workflow** → **Run workflow** をクリック
4. 数分待つと `docs/data.json` が自動生成されます

### Step 5: スマホでアクセス

- `https://<ユーザー名>.github.io/<リポジトリ名>/` をスマホで開く
- **「ホーム画面に追加」** でアプリ風に使えます（PWA対応）

### Step 6: 銘柄を変更したいとき

`watchlist.py` の `WATCHLIST` だけ編集してコミットしてください。
次回の自動実行（平日 16:00 JST）で反映されます。

## 自動更新スケジュール

- **毎日 16:00 JST**（UTC 07:00）に自動実行（平日のみ）
- GitHub Actions の `workflow_dispatch` で手動実行も可能

## テクニカル指標

| 指標 | パラメータ | 計算方法 |
|------|-----------|---------|
| 移動平均線 | 5 / 25 / 75日 | 単純移動平均（SMA） |
| ボリンジャーバンド | 20日, 2σ | mean ± 2×std(ddof=0) |
| MACD | 12 / 26 / 9 | EMA（ewm span, adjust=False） |
| RSI | 14日 | Wilder法（ewm alpha=1/14） |
| 出来高比率 | 20日平均比 | 当日出来高 / SMA(20) |

## スコアリング

| スコア | 判定 | アクション |
|--------|------|-----------|
| +5以上 | 🟢🟢 強い買い | エントリー検討 |
| +3〜+4 | 🟢 買い検討 | 押し目でエントリー候補 |
| +1〜+2 | 🔵 やや強気 | ウォッチ継続 |
| -1〜0 | ⚪ 中立 | 動きを待つ |
| -2〜-3 | 🟡 やや弱気 | 新規見送り |
| -4〜-5 | 🔴 売り検討 | 損切り検討 |
| -6以下 | 🔴🔴 強い売り | 即損切り |
